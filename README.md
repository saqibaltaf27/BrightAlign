# BrightAlign 24/7 - Emergency & Commuter Orthodontic Landing Page

**BrightAlign** is a night-focused orthodontic clinic offers appointments between **6:00 PM and 2:00 AM** and provides emergency orthodontic services, including quick wire fixes. This landing page highlights the clinic's services, provides an interactive **Commuter Value Calculator**, and emphasized the convenience of evening appointments.

## Features
1. Night Mode UI - Dark mode by default with neon teal and violet accents.
2. Commuter Value Calculator - Calculate time and productivity saved by scheduling night appointment.
3. Emergency Wire Fix Modal - Sticky button opens mobile-friendly modal for urgent services requests.
4. Service Tiers Section - Midnight Aligners, Commuter Braces, 2 AM Emergency
5. Responsive Design - Mobile-first layoutm optimized for commuters on phones or tablets.
6. Accessible HTML - Semantic markup, ARIA labels, alt text, and sufficient color contrast.
7. Performance Optimized - Images compressed for fast loading

## Setup and Running Locally

### Option 1: Open in Browser
1. Navigate to project folder.
2. Open `index.html` in your browser.

### Option 2: Use VS Code Live Server
1. Install the `Live Server` extension in VS Code.
2. Right Click `index.html` -> **Open with Live Server**

## Usage

### 1. Commuter Calculator
    - Enter your daily commute time in hours (e.g, 1)
    - Enter your hourly wage (e.g, 20)
    - Click `Calculate Savings` to see time and productivity saved.

### 2. Emergency Wire Fix
    - Click the Emergency Wire Fix sticky button.
    - Fillout the form and submit your urgent repair request.

### 3. Service Tiers
    - Midnight Aligners (Standard)
    - Commuter Braces (Fixed)
    - The 2 AM Emergency (Urgent Repair)

## Calculator Logic
Saved Hours = Commuter Time * 2
Value Saved ($) = Saved Hours * Hourly Wage
