# BrightAlign 24/7 - Emergency & Commuter Orthodontic Landing Page

## Overview
The goal of this project is to build a hihg-conversaion landing page that reflects the brand identity of a night clinic and provides useful tools for commuters. The page must visually represent a nightime aesthetic while highlighting the ubique services and extended hours offered by BrightAlign.

The landing page should include an interactive commuter value calculator which helps visitors determine how much productivity they gain by scheduling appointments after work instead of leaving work during daytime hours.

## Features
1. Night Themed UI - The website must default to dark mode. The color palette should use deep charcoal backgrounds combines with neon teal and violet highlights to create a nighttime medical-tech atmosphere.

2. Commuter Value Calculator - Users must be able to enter their commute time and hourly wage. The calculator should compute the amount of work time saved by scheduling a night appointment instead of leaving work during daytime hours. The calculator should display the total value of saved productivity.

3. Emergency Wire Fix Modal

4. Service Tiers (Standard, Fixed, Urgent Repair)

5. Responsive Design


## Deliverables
1. Project Codebase: A ZIP file containing all source files (HTML, CSS, JS, assets).
2. Public Repository: A link to a public GitHub repository containing the version-controlled code and a README.md with setup instructions.
3. Live Demo: A URL to the deployed landing page (e.g., Vercel, Netlify, or GitHub Pages).
4. Visual Documentation: A folder containing 3-4 screenshots (Desktop and Mobile views) and a brief (30-60 second) screen recording demonstrating the Commuter Value Calculator in action
